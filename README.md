# SpringBootDemo

#### SpringBoot2.x 整合Mybatis,Redis(含分布式锁实现/接口限流实现),Shiro(权限安全框架),消息队列功能实现(RabbitMQ/Kafka),Thymeleaf,Swagger2等简单案例

#### 一.项目运行要求
完整测试本Demo所有功能接口需要安装如下工具：

平台：Windows

1.Redis

2.Erlang + RabbitMQ

3.Mysql

4.Zookeeper+Kafka（高吞吐量、低延迟的分布式消息队列实现）

安装完成后启动，并在Demo中修改相关配置成你自己的

IDEA插件：

1.Lombok
